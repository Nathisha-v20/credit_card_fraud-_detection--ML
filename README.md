# Credit Card Fraud Detection

## Objective
To detect fraudulent transactions using Machine Learning.

## Dataset
Contains transaction details like amount, time, merchant category, and fraud label.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Workflow
1. Data Loading
2. Exploratory Data Analysis
3. Data Preprocessing
4. Model Training (Logistic Regression)
5. Model Evaluation

## Results
- Accuracy: ~99%
- Confusion Matrix and Classification Report used for evaluation

## Conclusion
The model successfully detects fraudulent transactions but shows limitations due to class imbalance.
